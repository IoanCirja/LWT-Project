# GoodReads Clone

## Step 1:

1. Relace `your_path_here` in `database\tables.txt` to your `database\` folder location
2. Copy `database\tables.txt` in Query Tool of your database and run it

## Step 2:

1. Add database credentials in `backend\db.js`

## Step 3:

1. cd .\backend
2. npm install
3. node .\index.js
